# Template Editor - Complete Guide

## Overview
The Template Editor is a powerful tool for Super Admins to create and customize templates that admins can use for their digital visiting cards. It's fully dynamic and allows complete control over every aspect of a template.

## Accessing the Template Editor

### Create New Template
1. Navigate to `/super-admin/templates`
2. Click "Create New Template" button
3. Fill in all sections
4. Click "Save Template"

### Edit Existing Template
1. Navigate to `/super-admin/templates`
2. Click the edit icon on any template card
3. Modify any fields
4. Click "Save Template"

## Template Editor Sections

### 1. Template Preview (Top Section)

**Real-time Preview Card:**
- Shows how the template will appear to admins
- Updates instantly as you make changes
- Displays:
  - Template icon and name
  - Category
  - Description
  - All features as colored badges
  - Color scheme (Primary, Secondary, Accent)
  - Layout style, font family, and section count

**Purpose:** Visual feedback to see your changes immediately

### 2. Basic Information

**Fields:**

**Template Name*** (Required)
- The display name of the template
- Example: "Salon & Spa", "Restaurant & Cafe"
- Should be descriptive and business-type specific

**Category*** (Required)
- Business category for grouping
- Example: "Beauty & Wellness", "Food & Beverage"
- Helps admins find relevant templates

**Icon**
- Visual identifier for the template
- Choose from 30+ emoji options
- Or enter custom emoji (2 characters max)
- Examples: 💇 (salon), 🍽️ (restaurant), 🏥 (medical)

**Display Order**
- Numeric value for sorting templates
- Lower numbers appear first
- Default: 1
- Use: 1, 2, 3... for ordering

**Description*** (Required)
- Brief explanation of template purpose
- 2-3 sentences recommended
- Example: "Perfect for beauty salons, spas, and wellness centers"

**Active Status**
- Checkbox to enable/disable template
- Active templates are visible to admins
- Inactive templates are hidden but not deleted

### 3. Theme & Design

**Color Customization:**

**Primary Color**
- Main brand color
- Used for headers, buttons, accents
- Color picker + hex input
- Example: #E91E63 (pink for salon)

**Secondary Color**
- Supporting color
- Used for hover states, backgrounds
- Should complement primary
- Example: #F06292 (lighter pink)

**Accent Color**
- Highlight color
- Used for important elements
- Should contrast with primary
- Example: #C2185B (darker pink)

**Typography:**

**Font Family**
- Choose from 15 professional fonts
- Options include:
  - Inter (modern, clean)
  - Playfair Display (elegant)
  - Roboto (professional)
  - Montserrat (bold)
  - Lora (classic)
  - And 10 more...

**Layout & Style Options:**

**Layout Style**
- Overall design approach
- 14 options: modern, classic, minimal, elegant, professional, creative, luxury, energetic, academic, corporate, portfolio, artistic, tech, traditional

**Header Style**
- Navigation bar appearance
- Options: clean, centered, sticky, bold, transparent, professional, modern, minimal, executive

**Card Style**
- Content card appearance
- Options: rounded, shadow, minimal, elevated, elegant, angular, clean, property, borderless, formal

**Button Style**
- Call-to-action button design
- Options: rounded, rounded-full, square, outlined, minimal, sharp, professional

### 4. Section Configuration

**Purpose:** Define which sections are available and their default styles

**Managing Sections:**

**Enable/Disable Sections:**
- Checkbox to toggle section availability
- Enabled sections appear in admin's card editor
- Disabled sections are hidden

**Default Style Selection:**
- Each section type has multiple style options
- Choose the default style for new cards
- Admins can change this later

**Available Sections & Styles:**

**Hero Section:**
- Styles: modern, classic, minimal, full-screen, video-background, image-overlay, centered, professional

**About Section:**
- Styles: two-column, centered, story, mission, credentials, expertise, institution, creative

**Services Section:**
- Styles: grid-cards, list-detailed, icons-grid, professional-list, cards, packages, categorized

**Products Section:**
- Styles: grid-shop, showcase, listings, featured, tiles

**Team Section:**
- Styles: circular-photos, professional-cards, profiles, academic

**Gallery Section:**
- Styles: masonry, grid, lightbox, masonry-full

**Contact Section:**
- Styles: inline, map-sidebar, appointment, booking, store-locator, inquiry, consultation, admission, membership

**Adding Custom Sections:**
1. Type section name in the input field
2. Press Enter or click "Add"
3. Section appears with default settings
4. Configure style and enable/disable

**Removing Sections:**
- Click trash icon next to section name
- Section is removed from template
- Can be re-added anytime

### 5. Template Features

**Purpose:** Define capabilities available to admins using this template

**Adding Features:**
1. Type feature name in input field
2. Press Enter or click "Add Feature"
3. Feature appears as a badge
4. Can add unlimited features

**Feature Examples:**
- Online Booking
- Service Menu
- Digital Menu
- Product Catalog
- Team Profiles
- Photo Gallery
- Before/After Gallery
- Appointment Booking
- Table Booking
- Virtual Tours
- Client Reviews
- Membership Plans

**Removing Features:**
- Click trash icon on feature badge
- Feature is removed
- Can be re-added anytime

**Best Practices:**
- Add 3-6 features per template
- Make features specific to business type
- Use clear, descriptive names
- Think about what makes this template unique

## Advanced Tips

### Color Selection
1. **Use color psychology:**
   - Blue: Trust, professionalism
   - Green: Health, nature
   - Red/Orange: Energy, food
   - Purple: Luxury, creativity
   - Black/Gray: Elegance, sophistication

2. **Ensure contrast:**
   - Primary should stand out
   - Secondary should complement
   - Accent should pop

3. **Test accessibility:**
   - Check color contrast ratios
   - Ensure text is readable

### Font Pairing
- **Elegant:** Playfair Display + Lora
- **Modern:** Inter + Roboto
- **Professional:** Merriweather + Open Sans
- **Creative:** Montserrat + Raleway
- **Bold:** Oswald + PT Sans

### Section Strategy
- **Essential:** Hero, About, Contact (always include)
- **Business-specific:** Services for service businesses, Products for retail
- **Optional:** Gallery, Team (based on business needs)
- **Custom:** Add unique sections for special requirements

### Feature Planning
- **Core features:** What every business in this category needs
- **Premium features:** Advanced capabilities
- **Unique features:** What sets this template apart
- **Future-proof:** Consider features businesses might need later

## Troubleshooting

### Template Not Loading
**Issue:** Edit page shows loading forever
**Solution:**
- Check browser console for errors
- Verify template ID in URL is correct
- Ensure Firestore rules allow read access
- Try refreshing the page

### Changes Not Saving
**Issue:** Click save but changes don't persist
**Solution:**
- Check internet connection
- Look for error messages
- Verify Firestore rules allow write access
- Check browser console for errors

### Template Not Appearing for Admins
**Issue:** Template saved but admins can't see it
**Solution:**
- Ensure "Active" checkbox is checked
- Verify template was saved successfully
- Check admin is logged in correctly
- Refresh admin's template selection page

### Colors Not Displaying Correctly
**Issue:** Colors look different than expected
**Solution:**
- Use hex format (#RRGGBB)
- Avoid transparency values
- Test in preview section
- Check color picker vs text input match
