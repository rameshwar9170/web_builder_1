# Feature System - User Guide

## Quick Overview

The feature system allows you to control which sections appear on your digital visiting card. Think of it like customizing a menu - you can show or hide items, but they're always available to add back.

## For Admins: Creating a Card

### Step 1: Choose Your Template
Pick a template that matches your business type (e.g., Restaurant, Salon, Medical).

### Step 2: Select Features ⭐ NEW!

You'll see all available features for your chosen template:

```
┌─────────────────────────────────────┐
│  🍽️ Restaurant & Cafe              │
│  Perfect for restaurants and cafes  │
│                                     │
│  [Enable All]  [Disable All]       │
└─────────────────────────────────────┘

┌──────────────────┐  ┌──────────────────┐
│ ✅ Digital Menu  │  │ ✅ Food Gallery  │
│ Visible          │  │ Visible          │
└──────────────────┘  └──────────────────┘

┌──────────────────┐  ┌──────────────────┐
│ ⬜ Online Orders │  │ ⬜ Table Booking │
│ Hidden           │  │ Hidden           │
└──────────────────┘  └──────────────────┘
```

**What to do:**
- Click any feature to toggle it on/off
- Green = Visible on your card
- Gray = Hidden (but not deleted!)
- Use "Enable All" to show everything
- Use "Disable All" to hide everything

**Pro Tip:** Start with all features enabled, then hide what you don't need yet.

### Step 3: Basic Information
Enter your business details and create your unique URL.

## Managing Features After Creation

### Opening Feature Manager

1. Go to your card's edit page
2. Click the **"Manage Features"** button (purple button in top toolbar)
3. A panel will open showing all your template features

### How Tabs Change

**All Features Enabled:**
```
[Basic Info] [About] [Services] [Products] [Team] [Gallery] [Theme]
```

**Some Features Disabled:**
```
[Basic Info] [About] [Services] [Team] [Theme]
```
(Products and Gallery tabs are hidden)

**Important:** Basic Info, About, and Theme tabs are ALWAYS visible.

## Common Scenarios

### Scenario 1: "I'm not ready for online booking yet"

**Solution:**
1. Open Feature Manager
2. Click "Online Booking" to turn it gray
3. Booking-related sections disappear
4. When ready, click it again to turn it green
5. Booking features are back instantly!

### Scenario 2: "I want to see what my card looks like with minimal features"

**Solution:**
1. Open Feature Manager
2. Click "Disable All"
3. Only essential tabs remain
4. Review your minimal design
5. Click "Enable All" to restore everything

### Scenario 3: "I accidentally disabled a feature I need"

**Solution:**
1. Open Feature Manager
2. Find the feature (it's still there, just gray)
3. Click it to turn it green
4. Feature is back immediately
5. No data was lost!

### Scenario 4: "I want to add a new feature later"

**Solution:**
1. All template features are always available
2. Open Feature Manager anytime
3. Enable the feature you want
4. Start adding content to that section
5. Publish when ready

## Best Practices

### ✅ Do's

1. **Start with everything enabled** - See all options first
2. **Hide what you don't need yet** - Keep your card focused
3. **Enable features as you grow** - Add booking when ready
4. **Experiment freely** - All changes are reversible
5. **Use "Disable All" to test** - See minimal version

### ❌ Don'ts

1. **Don't worry about losing data** - Features are never deleted
2. **Don't recreate cards** - Just enable/disable features
3. **Don't leave features enabled if empty** - Hide unused sections
4. **Don't forget to check preview** - See how it looks to visitors

## Troubleshooting

### "I don't see the Feature Manager button"
- Make sure you're on the edit page (not create page)
- Look for the purple "Manage Features" button in the top toolbar

### "No features available for this template"
- Your card was created before the template system
- You can still use the card normally
- To get features, create a new card with a template

### "I disabled a feature but the tab is still there"
- Basic Info, About, and Theme tabs are always visible
- Other tabs (Services, Products, Team, Gallery) hide when disabled

### "My changes aren't saving"
- Check your internet connection
- Look for success/error messages
- Try refreshing the page

## Tips for Different Business Types

### Restaurant/Cafe
- Enable: Digital Menu, Food Gallery

### Salon/Spa
- Enable: Service Menu, Team Profiles, Before/After Gallery
- Consider: Online Booking (when ready)

### Medical/Clinic
- Enable: Services List, Doctor Profiles
- Consider: Appointment Booking (when ready)

### Retail Store
- Enable: Product Catalog, Special Offers
- Consider: Store Location, Categories

### Professional Services
- Enable: Service Details, Experience Timeline
- Consider: Consultation Booking, Certifications
